# CyberSecurityBot

## 📌 Project Overview
CyberSecurityBot is a GUI-based chatbot designed to raise cybersecurity awareness.  
It integrates interactive features such as task management, quizzes, NLP simulation, and an activity log to make learning engaging and practical.

---

## 🎯 Features
- **Task Assistant**  
  Manage cybersecurity-related tasks (e.g., "Enable two-factor authentication").  
  - Add, view, delete, and mark tasks complete.  
  - Optional reminders for tasks.  

- **Cybersecurity Quiz Game**  
  - 10+ multiple-choice questions covering phishing, passwords, safe browsing, and social engineering.  
  - Immediate feedback after each answer.  
  - Final score summary at the end.  

- **NLP Simulation**  
  - Recognizes varied phrasing of user commands.  
  - Example: "Remind me to update my password" → Adds a reminder task.  

- **Activity Log**  
  - Records all major actions (tasks, reminders, quiz attempts).  
  - Displays the last 5–10 actions with timestamps.  

---

## 🖥️ GUI Layout
- **Header**: Banner with CyberSecurityBot title and logo.  
- **Chat Area**: Scrollable panel for bot and user messages.  
- **Controls**: Buttons for *Send*, *Start Quiz*, *Add Task*, *View Log*.  
- **Activity Log**: Displays recent actions.
