# CyberSecurityBot

A C# console-based Cybersecurity Awareness Bot that educates users on 
password safety, phishing, and safe browsing using voice greeting, 
ASCII art, and interactive menus.

## How to Run
1. Open `CyberSecurityBot.sln` in Visual Studio
2. Press `F5` or click Start to run

## Features
- Voice greeting on startup
- ASCII art display
- Interactive menu system
- Cybersecurity topics: passwords, phishing, safe browsing
- Input validation
- Typing effect and coloured UI
