# CyberSecurityBot 🤖

A C# Cybersecurity Awareness Chatbot that educates users on password safety, phishing, and safe browsing.

-----

## Part 1 — Console Application

A console-based chatbot with voice greeting, ASCII art, and interactive menus.

### Features

- 🔊 Voice greeting on startup
- 🎨 ASCII art display
- 📋 Interactive menu system
- 🔐 Cybersecurity topics: passwords, phishing, safe browsing
- ✅ Input validation
- ⌨️ Typing effect and coloured UI

### How to Run

1. Open `CyberSecurityBot.sln` in Visual Studio
1. Set **CyberSecurityBot** as startup project
1. Press `F5` or click Start to run

-----

## Part 2 — WPF GUI Application

An expanded version of CyberBot with a full Graphical User Interface.

### Features

- 🔊 **Voice Greeting** — Plays a WAV audio greeting on launch
- 💬 **Keyword Recognition** — Recognises cybersecurity topics using a Dictionary
- 🎲 **Random Responses** — Randomly selects phishing tips from a List
- 🔄 **Conversation Flow** — Handles follow-up questions naturally
- 🧠 **Memory and Recall** — Remembers user name and interests
- 😟 **Sentiment Detection** — Detects emotions and responds empathetically
- 🛡️ **Error Handling** — Gracefully handles unrecognised inputs

### How to Run

1. Open `CyberSecurityBot.sln` in Visual Studio
1. Set **PART 2** as startup project
1. Press `F5` to run
